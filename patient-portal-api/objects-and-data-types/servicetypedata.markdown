# ServiceTypeData

Information about the Service Type.

## Properties

| Parameter | Type   | Description                                                 |
|:----------|:-------|:------------------------------------------------------------|
| Key | string | Unique key of the service type |
| Name | string | Name of the service type |

## JSON Example

```
{

"Key": 84,

"Name": "Blood profile"

}
```
