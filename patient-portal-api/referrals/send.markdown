# Send

Sends the referral.

## JavaScript library method

```
patientportal.referrals.send({

referral: &lt;referral&gt;,

});
```

## HTTP Method

GET

## ****Url****

/patientportalapi/referrals/send

## URL Parameters

| referral | string | The key of the referral provided by the API upon GetReferrals. |
| --- | --- | --- |

## Returns

ReferralData
