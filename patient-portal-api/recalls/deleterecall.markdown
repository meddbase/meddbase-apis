# DeleteRecall

Removes the recall.

## JavaScript library method

```javascript
patientportal.recalls.deleteRecall({
    recall: <recall>
});
```

## HTTP Method

GET

## ****Url****

/patientportalapi/recall/delete

## URL Parameters

| recall | string | The key of the recall provided by the API upon method GetRecalls. |
